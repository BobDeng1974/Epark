# Epark快捷停车系统
##### 由小黄车app架构得到启发，刚好有开发一个停车app需求，于是就摸索着做了这个app
----------
## 主要功能
>* 用户注册<br>
>* 车位发布<br>
>* 车位查找<br>
>* 导航停车<br>
----------
## 提示
* 最新更新优化了自动布局，适配各款iPhone和iPad
----------
